## Käyttöohje
### esivaatimukset
- Go 1.24.1 tai uudempi
- Make (vapaaehtoinen)

### lataaminen
```bash
git clone https://github.com/Eeritvan/face_recognition.git
cd face_recognition
```

## ajaminen
```bash
#Käyttämällä make juurikansiosta:
make


# tai käyttämällä Go:
cd src
go run .
```
